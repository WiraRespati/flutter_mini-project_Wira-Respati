# StoryApp

## About Project
Aplikasi ini dibuat untuk dapat berbagi cerita antar user berupa description, gambar dan dimana story tersebut dibuat. 

## Features
### User
- Register
- Login 
- List Story
- Detail Story
- Story Location
- Upload Story
- AI Recommend
- Profile
